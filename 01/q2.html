Question 2: Single Handler with Capture vs Bubbling Phases

Instructions: Hover over any element on this page to highlight it. Use the button below to toggle between CAPTURE and BUBBLING phases. Watch the browser console (F12) to see which phase is active.

Analysis: For mouseover/mouseout events on interactive elements, which phase is more favorable?

Answer: BUBBLING phase is more favorable because:

  • More intuitive for user interactions (events bubble up from target)
  • Allows easier event delegation
  • Natural flow for mouseover/mouseout events
  • CAPTURE phase processes events too early before user interaction is complete
